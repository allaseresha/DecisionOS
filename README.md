# DecisionOS (MVP)

DecisionOS is an explainable, governance-first Decision Intelligence platform
designed for high-stakes business decisions.

## Core principles
- Explainability over black-box AI
- Auditability by default
- Explicit ownership & accountability
- Decision learning over time

## What this MVP demonstrates
- Deterministic scoring engine
- Governance readiness gate
- Explainable outcomes
- Scenario stress testing
- Versioned decision history

Built using Streamlit and a custom decision engine.
